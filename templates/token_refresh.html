Manual Authorization Steps:

  1. Click the "Generate New Auth Code" button below.
  2. A new tab will open. Log in with your Fyers credentials, enter PIN and TOTP as needed.
  3. After successful login, you will be redirected to a blank page with a URL like:
    {{ request.base_url.replace('token-refresh', 'token-callback') }}?code=YOUR_AUTH_CODE_HERE&state=...
  4. Copy only the 'code' parameter value from the URL (e.g., if the URL is `...code=YOUR_AUTH_CODE_HERE&...`, copy `YOUR_AUTH_CODE_HERE`).
  5. Paste the copied code into the input field below and click "Submit Auth Code".
